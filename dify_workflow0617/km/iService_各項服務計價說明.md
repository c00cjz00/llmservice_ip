# iService_各項服務計價說明

### Q: 臺灣AI雲服務(TWCC)計價 
A:
* 隨用隨付方案On-Demand：
彈性依您的成本與程式需求調配運算資源的多寡，隨時開始與停止。按使用量計費，用多少付多少，無需綁定長期合約。
* 以秒計費：
方案皆採以秒計費的模式計價，無需擔心用不滿一小時的部分將被收費，讓您的投資皆可精準的用在測試與開發。
* 詳細價目表請見：https://man.twcc.ai/@twccdocs/SJWlN3YDr?type=view


### Q: 台灣杉三號非保留佇列(NRQ)費率 
A:
台灣杉三號除專屬計算資源外亦提供非保留佇列NRQ (Non-Reserved Queue)服務，惟此服務不具優先序、必須排隊等候、且工作可能被中斷再重排等，請用戶確定能接受再行使用。
1. 非保留佇列NRQ：沒有固定租期，須依系統排程政策下排隊使用。因為此為臨時釋出之資源，且以公用方式提供，用戶須與他人共用計算資源。
2. NRQ被中斷再重排：若系統資源不足，則執行中NRQ工作因不具優先序，有可能被中斷。此時，該工作中斷前的使用量不會計費，且被中斷的工作將由系統自動派送重新排隊，此時此工作將具有優先序。
3. 費率：NRQ採固定費率計價，非學研界使用費率：凡政府、法人、企業、及個人等非學研界使用費率，為上述對應各費率之2倍。


|適用對象   | 優惠費率(元/核心小時) |
| -------- | -------- | 
| 國科會計畫/學術計畫   | 0.08   | 
| 政府法人/企業個人     | 0.16   |

### Q: 晶創25（NANO 5）非保留佇列(NRQ)費率 
A:
晶創主機之正式名稱為晶創25（英文：NANO 5），將於 2025年3月20日正式收費。晶創25主機為AI專用型計算主機，為生成式AI基礎模型之關鍵資源，有助於生成式AI技術研發與應用服務。主機規格相關內容，請參考 國網中心官網與iService系統服務介紹。

以下提供節點規格、收費方式、使用說明與注意事項，敬請用戶參考。

［使用額度與收費費率］
帳號之使用額度與收費標準皆以GPU小時為計算服務單位
GPU小時（GPU小時）=執行小時數 × GPU 數
［節點規格］每台計算節點規格：

 - 8 片 NVIDIA H100 GPU
 - 8 個 InfiniBand 400Gb/s 網路埠
 - 2TB 記憶體

［收費方式］各類計畫之費率：(單位：元)

| 學術計畫      | 50   |
